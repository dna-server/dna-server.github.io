Trash Magic Blueprint Library

This is a set of techniques and blueprints which have been collected to help individuals and communities live well. We want to replicate communities that prioritize connectedness, abundance, and meaning. Trash Magic is: (1) good for the community, (2) made for free, and (3) holds instructions to (a) copy, (b) change, and (c) recycle itself. Please read the guide below titled "WHAT IS TRASH MAGIC?"
