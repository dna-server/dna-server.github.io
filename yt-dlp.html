yt-dlp YOUTUBE DOWNLOADER

yt-dlp tutorial. There are several more tutorials online. Eventually these should be summarized onto this page for easer and simpler access.
you can use this program to download videos or music off of youtube, as well as vimeo and a few other sites which you can check at https://github.com/yt-dlp/yt-dlp/blob/master/supportedsites.md

after installation, open a terminal or command prompt and use the command:

yt-dlp -o "filename" URL

for example
yt-dlp -o "DesertSun02 - DIY Evap Air Cooler" https://www.youtube.com/watch?v=I6u6yCe8RzI

  • -x means audio only
  • -o sets output filename
  • yt-dlp -x --audio-format mp3 -o "artist name - song name" URL as in:
  • yt-dlp -x --audio-format mp3 -o "Grateful Dead - Touch of Grey" https://www.youtube.com/watch?v=mzvk0fWtCs0

if you receive an error "UNABLE TO EXTRACT UPLOADER ID", try this fix:
open the file found at:

/usr/lib/python3/dist-packages/yt_dlp/extractor/youtube.py (or wherever the "youtube.py" file has been installed on your system)

and use the "control" key + the "f" key (ctrl+f) to find the following line:

'uploader_id': self._search_regex(r'/(?:channel|user)/([^/?&#]+)', owner_profile_url, 'uploader id') if owner_profile_url else None,

change it so that instead it says:

'uploader_id': self._search_regex(r'/(?:channel/|user/|(?=@))([^/?&#]+)', owner_profile_url, 'uploader id', default=None),
